This is a reverse engineering tool for Java/Enterprise Java applications.

Reverse engineering for Perl applications - started, please write me if you need this feature today ...

It may be useful if you have a lot of undocumented source code.

See [HowToStart](HowToStart.md) and [Screenshots](Screenshots.md).


---


This is a small portfolio & hobby project. It isn't easy to use this tool today, but you may try.

Are you developing something like this ? - Write me please.
I think that is good to have a different approaches/attempts/efforts.


---


It allows to draw UML sequence-diagrams by using
[AmaterasUML](http://amateras.sourceforge.jp)
(Eclipse plug-in for drawing UML diagrams).

Also you can use it without AmaterasUML, but only for generation html views of filtered program trace.
