Useful if you have a lot of undocumented source code.

It allows to draw UML sequence-diagrams by using
[http://amateras.sourceforge.jp AmaterasUML]
(Eclipse plug-in for drawing UML diagrams).

You can use it without AmaterasUML, but only for generation html or xml views of filtered program trace.
